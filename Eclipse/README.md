# Debug 调试 Java 程序

## 在 Package Explorer 视图调试 Java 程序，操作步骤如下：

    鼠标右击包含 main 函数的 java 类

    选择 Debug As > Java Application

    该操作也可以通过快捷键来完成，快捷键组合为 Alt + Shift + D, J。

    以上操作会创建一个新的 Debug Configuration（调试配置） ，并使用该配置来启动 Java 应用。
