默认情况下 Eclipse 字符集为 GBK，但现在很多项目采用的是 UTF-8，这是我们就需要设置我们的 Eclipse 开发环境字符集为 UTF-8， 设置步骤如下：
在菜单栏选择 Window -> Preferences -> General -> Workspace -> Text file encoding，在 Text file encoding 中点击 Other，选择 UTF-8，如图所示：
